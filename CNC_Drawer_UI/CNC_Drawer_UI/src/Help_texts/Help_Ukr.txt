Com. COM-порт це порт, який створюється при підключенні контролера Arduino. Через нього здійснюється передача інформації на контролер.

ВАЖЛИВО! Перед натисканням на копку Start переконайтесь, що вибрали той COM-порт, який створила Arduino, подивитись це можна через: Панель керування => Диспетчер пристроїв => Порти(COM та LPT). Якщо такого немає або не підключена Arduino, або потрібно скачать драйвер, яких знаходиться в папці Driver, яку ви могли бачити при відкритті папки CNCDrawer в ній потрібно відкрити папку CH341SER та запустити Setup.exe, в якому натиснути Install.	

Browse. Кнопка Browse використовується для вибору зображення з комп'ютера. Бажано користуватись розширенням PNG, в іншому випадку можливі неточності чи помилки при обробці зображення.	
Zoom. Коефіцієнт збільшення зображення за рахунок збільшення довжини кожної лінії і відповідно збільшенню Ширини та Висоти зображення на цей коефіцієнт(Ширина*Zoom, Висота*Zoom).	
Filter. Коефіцієнт, який потрібен для більш точної обробки зображення. Так як на кожному зображені можуть бути сірі пікселі, які являють собою "шум", цей фільтр допомагає їх відкинути. Значення задає границі білого та чорного, тобто все що менше цього значення це чорний, а те що більше білий.	
Start. Початок малювання, який буде виконуватись до тих пір поки зображення малюється, для того щоб зупинити це потрібно закрити додаток.

ВАЖЛИВО! Перед натисканням на копку Start переконайтесь, що вибрали той COM-порт, який створила Arduino, подивитись це можна через: Панель керування => Диспетчер пристроїв => Порти(COM та LPT). Якщо такого немає або не підключена Arduino, або потрібно скачать драйвер, яких знаходиться в папці Driver, яку ви могли бачити при відкритті папки CNCDrawer в ній потрібно відкрити папку CH341SER та запустити Setup.exe, в якому натиснути Install.
