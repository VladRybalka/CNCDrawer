Com. COM-порт, який створюється при підключенні контролера Arduino. Через нього здійснюється передача інформації на контролер.

ВАЖЛИВО! Перед натисканням кнопки Start переконайтесь, що вибрали у випадаючому списку саме той COM-порт, який створила Arduino.
         Щоб переконатися:
             - підключіть Arduino до USB роз'єму комп'ютера
             - відкрийте Панель керування => Диспетчер пристроїв => Порти(COM та LPT).
             - знайдіть потрібний COM-порт
         Якщо COM-порт відсутній, потрібно встановити драйвер COM-порт.
         Для цього відкрийте папку CNCDrawer\Driver\CH341SER та запустіть Setup.exe і натисніть Install.	

Browse. Кнопка Browse використовується для вибору файлу зображення на комп'ютері. Бажано користуватись зображеннями у файлах з розширенням PNG. При  обробці зображень в інших форматах можуть виникати неточності чи помилки.	

Zoom. Коефіцієнт збільшення зображення. Масштабування здійснюється за рахунок збільшення довжини кожної лінії і відстані між лініями. Таким чином зростають розміри зображення = Ширина*Zoom, Висота*Zoom.	

Filter. Коефіцієнт, який потрібен для перетворення зображення з кольорового або сірого у чорно-біле. Змінюючи значення Filter, можна очистити білий фон від зайвих "шумових" пікселів і отримати чітке чорно-біле зображення. Коефіцієнт визначає межу білого та чорного. Все що менше цього значення вважається чорним кольором, а те що більше - білим.	

Start. Розпочати малювання. Процес продовжується до закінчення малювання зображення. Щоб перервати малювання потрібно закрити додаток, натиснувши кнопку X у верхньому правому куті вікна додатку.
